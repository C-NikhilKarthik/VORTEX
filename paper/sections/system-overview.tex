\section{System Overview}

The proposed framework is designed to enable efficient and robust 6-DoF tracking on mobile and embedded devices by combining hardware-aware feature processing with geometry-aware pose refinement. The system explores three progressively more capable tracking configurations: (i) a CPU-based algebraic tracking pipeline, (ii) a GPU-accelerated feature processing pipeline, and (iii) a geometry-aware render-and-track framework formulated on the $\mathrm{SE}(3)$ pose manifold. Figure~\ref{fig:backbone} illustrates the overall architecture of the system.

\input{figures/backbone}

All configurations share a common feature-based tracking backbone. Incoming frames are first converted to grayscale and processed using a FAST-based keypoint detector. Local binary descriptors are computed for the detected features and matched across frames using efficient Hamming-distance comparisons. These correspondences are then used to estimate camera motion through an algebraic alignment stage based on homography estimation, while lightweight optical flow tracking provides efficient frame-to-frame updates.

The first configuration executes the entire pipeline on the CPU and exploits SIMD vectorization to accelerate feature processing and descriptor matching. The second configuration introduces GPU acceleration by offloading keypoint detection and descriptor computation to programmable shaders, while descriptor matching is performed in parallel on the GPU. Both variants rely on the same algebraic tracking formulation.

The third configuration extends this pipeline with a geometry-aware pose refinement stage. Synthetic reference views of the known object are rendered from hypothesized poses, allowing image observations to be lifted to 2D--3D correspondences. Pose is subsequently refined through Gauss--Newton optimization on $\mathrm{SE}(3)$, enabling geometrically consistent tracking even under significant viewpoint changes.

Together, these three configurations enable a systematic comparison between purely algebraic tracking, hardware-accelerated feature pipelines, and geometry-aware pose estimation on mobile-class hardware.
